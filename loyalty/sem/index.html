Structural Equation Modeling (SEM)

SEM distinguishes between what you can observe (manifest variables) and what you can’t directly measure (latent variables) like true loyalty or satisfaction.

! Why This Matters

A customer who appears “loyal” based on transaction volume alone might actually be transactionally loyal — driven by convenience or lack of competition — and may defect as soon as a competitor opens nearby.

  • Emotional Loyalty: Brand affinity, genuine preference, resistant to competitor offers
  • Transactional Loyalty: Convenience-driven, price-sensitive, high defection risk
  • SEM uses simultaneous equations to model how different factors cause loyalty types
  • Path analysis shows causal relationships that simple models miss

SEM / DAG Workbench — Your Data, Your Variables, Your Model

A fully generic path-model builder. (1) Load data: upload any CSV — a generic 5-column dataset is pre-loaded so the tool works instantly. (2) Create your variables: add nodes to the empty canvas, then double-click a node to rename it and map it to a CSV column. (3) Draw the causal structure: click one node, then another, to draw a directed path; cycles are rejected so the drawn graph stays a recursive DAG and forms the structural matrix B. (4) Estimation runs automatically on every change: the sample covariance S is computed over exactly the columns you mapped, the drawn model’s implied covariance Σ is fit by maximum likelihood, arrows are annotated with standardized coefficients, and χ², df, and p report whether your DAG is consistent with the data.

4 · Goodness of Fit (Maximum-Likelihood χ²)

Model Comparison (Δχ² Test)

Save the current model as Model A, then change the structure — the live fit is always Model B. If one model is a simplified version of the other (same mapped variables, a subset of its paths, no variable switching exogenous/endogenous status), the nested difference test applies: Δχ² = χ²restricted − χ²general, Δdf = dfrestricted − dfgeneral. Non-nested pairs are compared with AIC/BIC.

Spoiler: how the default dataset was generated

400 rows simulated from: Var2 = 0.30·Var1 + e, Var3 = 0.60·Var1 + e, Var4 = 0.50·Var3 + 0.35·Var2 + e, Var5 = 0.40·Var4 + 0.20·Var1 + e, with Var1 ~ N(0,1) and all variables scaled to unit variance. Try recovering this structure — then delete a path (say Var2 → Var4) and watch χ² blow up.

Strategic Implication

Different loyalty types require different interventions. Transactionally loyal customers need barriers to switching; emotionally loyal customers need brand reinforcement — not discounts that might cheapen the relationship.
