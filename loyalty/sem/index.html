Structural Equation Modeling (SEM)

SEM distinguishes between what you can observe (manifest variables) and what you can’t directly measure (latent variables) like true loyalty or satisfaction.

! Why This Matters

A customer who appears “loyal” based on transaction volume alone might actually be transactionally loyal — driven by convenience or lack of competition — and may defect as soon as a competitor opens nearby.

  • Emotional Loyalty: Brand affinity, genuine preference, resistant to competitor offers
  • Transactional Loyalty: Convenience-driven, price-sensitive, high defection risk
  • SEM uses simultaneous equations to model how different factors cause loyalty types
  • Path analysis shows causal relationships that simple models miss

Causal Path Workbench — Draw Causes, Read Off Correlations

You never type a correlation here. (1) Draw the causal model: create variables, draw directed paths, and type the causal coefficient directly on each path. (2) The engine computes the implied correlation matrix from your drawing via the RAM formulation Σ = (I − B)−1Ψ(I − B)−T, with residual variances set so every variable has unit variance — so your typed values are standardized coefficients and path-tracing algebra holds exactly. The matrix is a read-only output: watch variables with no arrow between them become correlated anyway through shared causes. (3) Designate a target to get every variable’s total causal effect on it, and save your model as Model A to test any rival structure you draw next: the rival is fitted by maximum likelihood against Model A’s implied matrix and judged with χ², df, and p.

4 · Goodness of Fit — Test a Rival Model

Save the current drawing as Model A (the “true” model) — its implied matrix becomes the data. Then redraw the structure however you like: the live drawing is Model B, and its structure is re-estimated by maximum likelihood against Model A’s matrix. A significant χ² means Model B’s structure cannot reproduce the world Model A describes.

Try it in 60 seconds: the donut confounder

Create three variables and rename them summer, donut_sales, and C (the confounder — say, boardwalk foot traffic). Draw C → summer and type 0.80 on it, then C → donut_sales at 0.80. The implied matrix now shows corr(summer, donut_sales) = 0.64 in amber — correlated with no arrow between them, purely through C. Mark donut_sales as the target: summer’s total causal effect on it is 0 (association ≠ causation). Now save this as Model A, delete both paths, draw summer → donut_sales directly, and watch the fit test reject the rival: χ²(1) ≈ 98 at n = 200, p < .001.

Strategic Implication

Different loyalty types require different interventions. Transactionally loyal customers need barriers to switching; emotionally loyal customers need brand reinforcement — not discounts that might cheapen the relationship.
